Warum gibt es Migration?

Zusammenwirken wirtschaftlicher, politischer und sozialer Faktoren

z.B. Chancen am Arbeitsmarkt, politische Stabilität, Meinungsfreiheit

Migration gab es immer schon!

z.B. National Geographic Genographic Project

Wie wird Migration definiert?

Stock: im Ausland geborene Bevölkerung

Flow: Wanderungsströme über einen bestimmten Zeitraum


Empfehlung der UNO zur Definition von Wanderungsströmen:

"Eine Person die für mindestens ein Jahr in ein anderes Land zieht, so dass das Zielland das neue gewöhnliche Aufenthaltsland ist."

Wie wird Migration erfasst?

Stock: in Volkszählungen (Frage nach dem Geburtsland)

Flow: in Bevölkerungsregistern und administrativen Daten

Die Erfassung von Wanderungsströmen ist sehr schwierig!

Es gibt keine international einheitliche Definition.

Erfassung von Migrationsströmen variiert von Land zu Land.

Vergleich nationaler Daten ähnelt dem zwischen Äpfeln und Birnen.

Harmonisierte Daten nur für Europa, z.B. IMEM.

Migrationsindikatoren für die Schweiz (2013)

1) Einwanderung von Ausländern: 176.000

2) Auswanderung von Ausländern: 77.000

3) Wanderungssaldo: 89.000 (1,8% der Bevölkerung)

4) im Ausland geborene Bevölkerung: 1,35 Millionen (19% der Bevölkerung)

5) Bevölkerung mit Migrationshintergrund: 2,37 Millionen (35% der Bevölkerung)

Quelle: Statistisches Bundesamt
